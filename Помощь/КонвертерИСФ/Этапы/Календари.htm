Общие сведения о конвертации

Конвертация производственных календарей

В реализации ИСФ таблица производственных календарей пополнялась ежегодно, идентификация типов производственных календарей осуществлялась по коду календаря. Поэтому список календарей ИСФ представляет собой список уникальных кодов, а не список записей.

Для конвертации необходимо каждому коду ИСФ сопоставить запись типа производственного календаря. При развертывании системы "Зарплата и ДД" типы производственных календарей автоматически НЕ создаются. Разработчик поставляет только ШАБЛОНЫ календарей. Необходимо создать рабочие производственные календари самостоятельно. При этом на этапе конвертации нет необходимости рассчитывать (заполнять) календари для конкретного года.

Календарю ИСФ с кодом "ДЕТИ" тип производственного календаря в п/с "Зарплата и ДД" не сопоставляется.
